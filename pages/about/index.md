# 私について

こんにちは、hoimohuと申します。ShoboiNamaeは別名義です。(ほかにもいろいろ名前があります)

競技プログラミングで強くなるために絶賛精進中です。ShoboiNamaeという名前は初めてAtCoderに登録したときにいい名前が思いつかなかったので適当につけた名前です。ずっとしょぼいといわれるのも嫌なので、強くなったら **TsuyoiNamae** とかに改名したいです。

hoimohuは小学生のころ自由帳に書いていた漫画にでてくるキャラクター2体の名前からとりました。

絵は下手くそなので上手くなりたいです。自前でキャラクターとか描きたい…
