# 更新内容

ブログの生成エンジンをだいぶ改良しました。

## コードをだいぶ見やすくした

これまでのコードは大部分がGithub Copilot様に頼りきりで、うまく使いこなせておらず結果的に全体がきたなくなってしまったのですが、今回ほぼすべてのコードを整えてみたので多少は読みやすくなったと思います。

## npm run dev したときに毎回全部上書きするのをやめた

これが実は一番したかったことで、これができたので満足です。他はおまけ。

## 変数としてファイルを参照できるようにした

これは別にやらなくてもよかったのですが、毎回AiScriptを使いまわしたいときにレイアウトに挿入できるようにする目的で入れてみました。多分いつか役に立つでしょう。

ただ、その過程でpageList変数の機能が弱体化することとなりました。今まではそれぞれのページのほとんどすべての情報を変数に入れていたのですが、AiSON.stringifyで `.` （ドット）が入った名前がうまく変換されず、冷静に考えたらこんなに情報いらないと思ったのでだいぶ情報を削りました。今のブログに影響はたぶんないです。

あと、 `.md` と `.ais` のファイルを指定すると内容がパーサーを通るようにしました。これを使ってOGPのmetaタグを生成するようにしてみたり。

## その他

あとはブログのエンジン以外の部分もだいぶ変更しました。また、いちいちブランチ切るのをめんどくさがってしまったので、一回のコミットで一気に更新します。（よくない）
