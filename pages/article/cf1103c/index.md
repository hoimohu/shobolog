# 結果

AB2完

# 感想

まず、久々に日本語以外の問題を解いたので読むのが大変でした。あと、問題文に出てくる人名や地名に慣れていなかったのでさらに大変でした。コンテスト中、Omskってなんだよってずっと思っていました。（Omskの人に失礼）

A, B問題は問題文が読めてしまえば簡単で、時間はかかりましたが正解することができました。

問題はCです。問題文は読めたのですが、ぱっといい解法が思いつかず、嘘貪欲をかいてWAを出しました。そして睡眠欲に負けて寝ました。

# コンテスト後

一応、コンテスト後にCは通しました。

xが2以上だったので30回割ったら10^9は0になりそうだと分かったのと、割るより先に1足す操作をするより先に割ったほうが回数が少なくできそうだと思ったので、a, bそれぞれに対して割る操作をする回数を0~30の間で決めて30*30のループを回し、割った後のa, bの差の絶対値と割った回数と足したものの最小値を探索しました。

# 最後に

変な貪欲を無理やり考えようとして時間が経つことが多いので実は貪欲をあまり考えない方がいいのではと思ったりしました。

あと、問題の解き方を考えるときにそれっぽいものを思いついても正しいことの証明が分からないことが多いのはどうすればいいのやら。
